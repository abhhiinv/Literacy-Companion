# Literacy Companion 

An AI-powered interactive literacy app built to support **UN SDG 4.6** — eliminating illiteracy among adults. Users read adaptive stories generated by AI, listen to narration, and take comprehension quizzes, all tailored to their reading level.

🌐 **Live App:** [https://literacy-companion.web.app](https://literacy-companion.web.app)

---

## Features

- **Adaptive Story Engine** — Generates short, adult-appropriate stories at Beginner, Intermediate, and Advanced levels using Gemini-2.5-flash-lite.
- **Comprehension Quizzes** — Auto-generated 3-question multiple-choice quizzes after every story.
- **Text-to-Speech** — Narrates the full story or individual words using the Web Speech API.
- **Word-Level Interaction** — Click any word to hear its pronunciation with a visual highlight.
- **Authentication** — Email/Password and Google Sign-in via Firebase Auth.
- **Progress Tracking** — Daily learning streaks and reading level saved to Firestore.
- **PWA Support** — Installable on mobile and desktop with offline capability.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19 (TypeScript) + Vite 8 |
| Styling | Bootstrap 5 + React-Bootstrap + React Icons |
| AI Engine | gemini-2.5-flash-lite|
| Backend | Firebase (Auth + Firestore) |
| Offline | Service Workers (Vite PWA) + Firestore Offline Persistence |

## Environment Variables

Create a `.env` file in the root of the project with these settings:

```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

> Never commit your `.env` file. It is already excluded via `.gitignore`.

## Deployment

This app is deployed on **Firebase Hosting**.

```bash
firebase deploy
```

Make sure your Firestore security rules allow read/write on `/users/{userId}` for authenticated users.

## Getting Started

### Prerequisites

- Node.js v18 or higher
- A Firebase project with Authentication and Firestore enabled
- A Google AI Studio API key for Gemini

### Clone the Repository

```bash
git clone https://github.com/abhhiinv/Literacy-Companion.git
cd Literacy-Companion
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root of the project and add the following:

```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

> Never commit your `.env` file. It is already excluded via `.gitignore`.

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
